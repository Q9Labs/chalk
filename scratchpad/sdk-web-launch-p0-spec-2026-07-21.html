System boundaries

Every piece has one authority

Go API

Owns customer authentication, participant credential issuance, exact route checks, and the Cloudflare secret.

Sync v3

Owns admitted membership, roles, moderation, live publications, and durable Leave.

Media adapter

Owns what the browser and peer connection actually published or received.

ChalkSession

Coordinates access, permission, both engines, recovery, commands, diagnostics, and cleanup.

React

Subscribes to a snapshot and exposes actions. It owns no networking or meeting policy.

First-party web

Exercises the same public package boundary a customer installs.

Failure contract

Recovery never broadens authority or fakes success

reject

Wrong credential family

A recognized API, Sync, or media credential is checked only by its intended verifier. There is no fallback.

recover

Sync disconnects

The Session stays durably present, refreshes access when needed, and reconnects within a bounded budget.

rebuild

Media peer fails

Fresh access creates a new Cloudflare connection, restores live local tracks, and reconciles remote state.

clean

Remote publication closes

The API records removal, the browser stops the old track, and the next snapshot no longer contains it.

finish

Leave times out

Local teardown still completes and exposes the timeout as a stable diagnostic instead of leaking resources.
