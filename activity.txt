# React Native SDK - End-to-End Verification Activity Log

## Setup Requirements (MUST be running before verification)

1. **Metro bundler**: cd packages/sdk-react-native/example && bun start
2. **iOS Simulator**: Must be booted (xcrun simctl list devices | grep Booted)
3. **IDB PATH**: Ensure ~/bin/idb exists or PATH includes Python bin directory

## iOS Simulator Skill Usage

Scripts are located at: ~/.claude/skills/ios-simulator/scripts/

Key commands:
- Launch app: python3 scripts/app_launcher.py --launch com.q9labs.chalk-example
- Tap by text: python3 scripts/navigator.py --find-text "Button Text" --tap
- Grant permissions: python3 scripts/privacy_manager.py --bundle-id com.q9labs.chalk-example --grant camera,microphone
- Screenshot: xcrun simctl io booted screenshot /path/to/output.png
- Map screen: python3 scripts/screen_mapper.py --verbose

Always prefix python scripts with: PATH="$HOME/bin:$PATH"

## Metro Log Location

Logs are at: /tmp/metro-output.log
Use: tail -50 /tmp/metro-output.log to check recent output
