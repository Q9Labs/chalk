```
Audio Flow in React Native WebRTC:
1. RTK receives remote tracks from SFU
```

**Key Difference from Web SDK**:
- Web SDK: Requires AudioRenderer component with explicit <audio> elements
- React Native: Audio auto-plays, just need AudioSession for routing/focus
