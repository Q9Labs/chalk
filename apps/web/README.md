# Chalk web

This Vite, React 19, and TanStack Router app currently serves three roles:

- the public marketing shell at `/`;
- a localhost-only API, Cloudflare SFU, and Sync v3 proof room at `/room`;
- a state-driven React SDK component preview at `/sdk-preview`.

It is not yet a complete hosted meeting product. `/new`, `/whiteboard`, `/status`, `/privacy`, and `/terms` are blank routes, and the SDK preview uses local fixtures rather than live room state. Track those gaps in the root [`checklist.md`](../../checklist.md).

Routes live in `src/routes/` and are generated by TanStack Router. Shared product behavior belongs in the SDK packages; this app should remain a thin first-party integration and verification surface.

## Commands

Run commands from the repository root:

```bash
pnpm --filter @q9labsai/chalk-web dev
pnpm --filter @q9labsai/chalk-web test
pnpm --filter @q9labsai/chalk-web build
```
