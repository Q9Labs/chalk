# SDK React

The `@q9labs/chalk-react` package provides React hooks and components for building video conferencing experiences. It wraps the core SDK with React-friendly APIs and includes pre-built UI components.

## Installation

```bash
npm install @q9labs/chalk-react
```

Import the styles in your app:

```tsx
import "@q9labs/chalk-react/styles.css";
```

---

## ChalkProvider

Wrap your application with `ChalkProvider` to enable Chalk features:

```tsx
import { ChalkProvider } from "@q9labs/chalk-react";

function App() {
  return (
    <ChalkProvider
      apiUrl="https://chalk-api.q9labs.ai"
      tokenProvider={async () => {
        // Return fresh access token
        const res = await fetch("/api/chalk/refresh");
        const { accessToken } = await res.json();
        return accessToken;
      }}
    >
      <YourApp />
    </ChalkProvider>
  );
}
```

### Provider Props

| Prop            | Type                    | Description                                   |
| --------------- | ----------------------- | --------------------------------------------- |
| `apiUrl`        | `string`                | Chalk API URL                                 |
| `wsUrl`         | `string`                | WebSocket URL (optional, derived from apiUrl) |
| `tokenProvider` | `() => Promise<string>` | Function to get fresh tokens                  |
| `debug`         | `boolean`               | Enable debug logging                          |

## Component Tiers

Chalk React provides three tiers of components for different levels of customization:

### Tier 0: Turnkey Components

Drop-in components for the fastest implementation. Zero configuration required.

```tsx
import { VideoConference } from "@q9labs/chalk-react";

function Room() {
  return <VideoConference roomId="my-room" userName="John Doe" authToken={token} />;
}
```

**Available turnkey components:**

- `VideoConference` - Complete meeting experience
- `MeetingRoom` - Full room with all controls
- `PreJoinLobby` - Device selection before joining
- `EndScreen` - Post-meeting screen
- `LoadingScreen` - Connection loading state

### Tier 1: Composable Components

Mix and match components to create custom layouts:

```tsx
import { VideoGrid, ControlBar, ChatPanel, ParticipantList, SettingsPanel } from "@q9labs/chalk-react";

function CustomMeeting() {
  return (
    <div className="flex h-screen">
      <div className="flex-1 flex flex-col">
        <VideoGrid />
        <ControlBar />
      </div>
      <aside className="w-80 border-l">
        <ChatPanel />
      </aside>
    </div>
  );
}
```

**Available composable components:**

- `VideoGrid` - Participant video layout
- `ControlBar` - Mute, share, record controls
- `ChatPanel` - Message list and input
- `ParticipantList` - Attendee roster
- `DeviceSelector` - Camera/mic picker
- `ScreenShareView` - Screen share display
- `NotificationStack` - Toast notifications
- `SettingsPanel` - Meeting preferences
- `RecordingControls` - Start/stop recording
- `WaitingRoom` - Host admission UI

### Tier 2: Atomic Components

Build fully custom UIs with low-level primitives:

```tsx
import { VideoTile, Avatar, ControlButton, NameTag, AudioIndicator } from "@q9labs/chalk-react";

function CustomVideoTile({ participant }) {
  return (
    <div className="relative rounded-lg overflow-hidden">
      <VideoTile participantId={participant.id} />
      <div className="absolute bottom-2 left-2 flex items-center gap-2">
        <Avatar name={participant.displayName} size="sm" />
        <NameTag>{participant.displayName}</NameTag>
        <AudioIndicator participantId={participant.id} />
      </div>
    </div>
  );
}
```

**Available atomic components:**

- `VideoTile` - Single participant video
- `Avatar` - User avatar
- `ControlButton` - Icon button
- `NameTag` - Participant label
- `AudioIndicator` - Volume meter
- `HandRaiseIndicator` - Raised hand badge
- `Tooltip` - Info popover

## Hooks Overview

Hooks are organized by category for easy discovery:

### Room Management

```tsx
import { useRoom, useConnection } from "@q9labs/chalk-react";

function RoomStatus() {
  const { status, roomId, join, leave } = useRoom();
  const { isConnected, error } = useConnection();

  return (
    <div>
      <p>Status: {status}</p>
      <p>Connected: {isConnected ? "Yes" : "No"}</p>
      {error && <p className="text-red-500">{error}</p>}
    </div>
  );
}
```

### Participants

```tsx
import { useParticipants, useActiveSpeaker } from "@q9labs/chalk-react";

function ParticipantCount() {
  const { participants, count } = useParticipants();
  const { activeSpeaker } = useActiveSpeaker();

  return (
    <div>
      <p>{count} participants</p>
      {activeSpeaker && <p>Speaking: {activeSpeaker.displayName}</p>}
    </div>
  );
}
```

### Media & Devices

```tsx
import { useMedia, useDevices, useScreenShare } from "@q9labs/chalk-react";

function MediaControls() {
  const { isVideoEnabled, isAudioEnabled, toggleVideo, toggleAudio } = useMedia();
  const { devices, selectedCamera, selectCamera } = useDevices();
  const { isScreenSharing, startScreenShare, stopScreenShare } = useScreenShare();

  return (
    <div className="flex gap-2">
      <button onClick={toggleAudio}>{isAudioEnabled ? "Mute" : "Unmute"}</button>
      <button onClick={toggleVideo}>{isVideoEnabled ? "Stop Video" : "Start Video"}</button>
      <button onClick={isScreenSharing ? stopScreenShare : startScreenShare}>{isScreenSharing ? "Stop Share" : "Share Screen"}</button>
    </div>
  );
}
```

### Features

```tsx
import { useChat, useRecording, useInteractions, useWhiteboard } from "@q9labs/chalk-react";

// Chat
const { messages, sendMessage, unreadCount } = useChat();

// Recording
const { isRecording, startRecording, stopRecording } = useRecording();

// Interactions
const { handRaised, raiseHand, lowerHand, sendReaction } = useInteractions();

// Whiteboard
const { isEnabled, data, updateWhiteboard } = useWhiteboard();
```

### UI State

```tsx
import { useLayout, usePanels, useNotifications } from "@q9labs/chalk-react";

// Layout modes: 'grid' | 'spotlight' | 'sidebar'
const { mode, setLayoutMode } = useLayout();

// Panel management
const { activePanel, openPanel, closePanel } = usePanels();

// Notifications
const { notifications, addNotification, dismissNotification } = useNotifications();
```

## Hook Reference

| Hook                   | Returns           | Description                            |
| ---------------------- | ----------------- | -------------------------------------- |
| `useChalk`             | Quick access      | Convenience hook for common operations |
| `useRoom`              | Room state        | Join, leave, room info                 |
| `useConnection`        | Connection state  | isConnected, error, status             |
| `useParticipants`      | Participants      | List, count, getParticipant            |
| `useActiveSpeaker`     | Active speaker    | Currently speaking participant         |
| `useMedia`             | Media controls    | Toggle audio/video                     |
| `useDevices`           | Device management | Camera, mic, speaker selection         |
| `useScreenShare`       | Screen sharing    | Start, stop, track                     |
| `useChat`              | Messaging         | Send, receive, react                   |
| `useRecording`         | Recording         | Start, stop, status                    |
| `useInteractions`      | Interactions      | Hands, reactions                       |
| `useWhiteboard`        | Whiteboard        | Collaborative drawing                  |
| `useLayout`            | Layout            | Grid, spotlight modes                  |
| `usePanels`            | Panels            | Side panel management                  |
| `useNotifications`     | Notifications     | Toast messages                         |
| `useSoundEffects`      | Audio             | Play join, leave sounds                |
| `useKeyboardShortcuts` | Shortcuts         | Keyboard bindings                      |

## TypeScript Support

All hooks and components are fully typed:

```tsx
import type { Participant, ChatMessage, Recording, RoomState, MediaState } from "@q9labs/chalk-react";

// Full type inference
const { participants } = useParticipants();
// participants: Participant[]

const { messages } = useChat();
// messages: ChatMessage[]
```
