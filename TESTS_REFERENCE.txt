EXTENDING THE TESTS
================================================================================

When adding new features:

1. Create test file: src/__tests__/<module>.test.ts
2. Import: import { describe, it, expect, beforeEach } from 'bun:test'
3. Use existing patterns for mocking and assertions
4. Run: bun test to verify
5. Type check: bun run check-types

Example structure:
  describe('Module', () => {
    let instance: MyClass;
    
    beforeEach(() => {
      instance = new MyClass();
    });
    
    describe('feature', () => {
      it('should do something', () => {
        expect(instance.method()).toBe(expected);
      });
    });
  });
